## ⚠️ ESCALATION PROCEDURE

### **Si Lighthouse < 85 en Jour 2 :**

```
1. Identifier bottleneck:
   ☐ Images trop lourdes → Récomprimer
   ☐ CSS > 50 Ko → Vérifier PurgeCSS
   ☐ Fonts bloquant → Ajouter font-display
   ☐ Contenu lourd → Lazy-load sections

2. Actions correctives:
   - Dev Lead + Frontend Dev 1: Diagnostic (15 min)
   - Implémentation fixes (30-45 min)
   - Re-test Lighthouse

3. Contingency:
   - Réduire contenu si nécessaire
   - Repousser features Day 4 vers Day 3+
   - Étendre Day 2 sur soirée si critique
```

### **Si Formulaires ne s'envoient pas en Jour 3 :**

```
1. Vérifier:
   ☐ Backend endpoint accessible
   ☐ CORS headers corrects
   ☐ Validation côté serveur
   ☐ Email service (SendGrid) actif

2. Fixes rapides:
   - Activer logs API
   - Tester manuelle via Postman
   - Vérifier honeypot ne bloque pas

3. Escalade:
   - Si > 30 min bloqué
   - Contactez backend support
   - Plan fallback: formulaire email temporaire
```
